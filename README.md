
# POKEMÓN MEMORY CARD GAME 🎮🃏

Pokémon Memory Game es una práctica desarrollada en JavaScript basada en el clásico juego de memoria.
El objetivo es encontrar todas las parejas de cartas Pokémon volteándolas de dos en dos hasta completar el tablero.

El proyecto incluye diferentes modos de juego, personalización de cartas y almacenamiento de récords usando Local Storage.

## Características principales ✨

- 🧠 Juego clásico de memoria con cartas Pokémon.
- 🔄 Animación de cartas giratorias.
- 🏆 Sistema de récords guardados con Local Storage.
- Parejas encontradas.
- Número de clicks realizados.
- Mejores puntuaciones.
- 📐 Posibilidad de cambiar el tamaño de la cuadrícula.
- 🎨 Personalización del diseño/modelo de las cartas.

## Tecnologías utilizadas 🛠️

- HTML5
- CSS3
- JavaScript

## Funcionamiento del juego 🎯
1. Se muestran todas las cartas boca abajo.
2. El jugador debe hacer click en dos cartas.
3. Si las cartas coinciden:
- Permanecen visibles.
- Se registra la pareja encontrada.
4. Si no coinciden:
- Las cartas vuelven a girarse.
- El juego termina cuando todas las parejas han sido encontradas.
